About {{ title }}


{{ title }} is a software platform to manage active traffic cameras. It is developed by the group sw803f20 on Aalborg Univerity as a student project.


Features

All the features of this webpage are accessible from the main page.


Viewing live streams from traffic cameras

On the main page, you can watch live streams from all cameras that are active and you are given access to. Make sure to be logged in to gain your access rights.


Recording video from traffic cameras

You can choose a live stream and you can then record a video of a set length. The video recorded will be available for download for a limited time on the main page after the recording has completed.

You can only have one recording active at a time per camera.


Managing cameras

Cameras can be set up and the settings of current cameras can be adjusted by accessing the drop-down menu for the selected camera.


A camera can be deleted by accessing the drop-down menu for the camera on the main page.


As a user with rights to set up new cameras, this can be done on the main page. You must give some information about the camera and you can give permissions to view the video feed to individual users.


You can update the machine learning model of the near-camera devices by selecting "Change Model" for a camera. Then upload the model file, and the near-camera device will change to the new model.


Adjusting global settings

As an administrator the global settings of the system can be adjusted by accessing the settings menu on the main page.
